## 需求

## API 規格
- URL: https://www.twse.com.tw/exchangeReport/BWIBBU
- 參數: response=json, date=YYYYMMDD, stockNo=股票代號
- stat == "OK" → 成功
- fields: ["日期","殖利率(%)","股利年度","本益比","股價淨值比","財報年/季"]
- data: list of list，日期 ROC 格式

## 欄位對應

| per_data | TWSE fields | 換算 |
|----------|-------------|------|
| stock_id | （傳入）| 直接 |
| date | 日期 | ROC→CE |
| dividend_yield | 殖利率(%) | float |
| per | 本益比 | float |
| pe_ratio | 本益比 | float（同 per）|
| pbr | 股價淨值比 | float |
| pb_ratio | 股價淨值比 | float（同 pbr）|
| source | — | 寫死 'official' |

注意：per 和 pe_ratio 寫同一個值；pbr 和 pb_ratio 寫同一個值。
