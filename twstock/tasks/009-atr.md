## 需求
實作 ATR14 計算模組：
- 讀取 stock_history 的 high、low、close
- 計算 True Range（TR）
- 用 Wilder's EMA（period=14）計算 ATR14
- UPSERT 寫入 stock_indicators.atr14（不動其他欄位）
- 前 13 天（TR 不足 14 個）atr14 = NULL

## 計算公式
TR(t) = max(high-low, |high-prev_close|, |low-prev_close|)
TR(1) = high(1) - low(1)（第一天無 prev_close）
ATR14(14) = mean(TR1..TR14)（第一個值用 SMA）
ATR14(t)  = (ATR14(t-1) * 13 + TR(t)) / 14（Wilder's EMA）
前 13 天 = NULL
