## 資料流向

```
外部 API  ──→ fetcher.py ──→ processor.py ──→ SQLite
SQLite  ──→ calculator.py ──→ strategy/*.py ──→ display.py
SQLite  ──→ strategy_runner.py ──→ JSON output
```

## 關鍵決策

| 決策 | 選擇 | 原因 |
|------|------|------|
| 資料庫 | SQLite | 輕量、零設定、單人專案 |
| 資料單位 | 張 | 台股慣例 |
| 價格 | 原始價（無復權） | 簡化計算、避免除權息調整複雜度 |
| 配色 | 漲紅跌綠 | 台股慣例 |
| 介面 | Rich Console + React | TUI + Web 雙平台 |
