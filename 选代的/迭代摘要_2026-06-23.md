# Dashboard 改版迭代記錄

**日期**: 2026-06-23

## 任務目標

1. 移除 Dashboard 舊版區塊（主要資訊、配股配息、其他分組、漲跌分布、櫃買指數）
2. 新增 4 個新查詢排行榜（1x4 橫排）
3. 新增資料庫健康檢查 API
4. 新增資料來源選擇器（本地/Supabase）

## 修改檔案

### 新增檔案
- `twse-app/src/server/routes/dashboard.ts` — 4 個新查詢端點 + DB 健康檢查
  - `GET /api/dashboard/recent-dividend` — 最近一週內準備發放股利
  - `GET /api/dashboard/foreign-buy-2day` — 投信連買二日（依成交量排）
  - `GET /api/dashboard/break-ma200` — 突破 MA200（昨日 < MA200 且 今日 > MA200）
  - `GET /api/dashboard/limit-up-yesterday` — 昨日涨停（漲幅 >= 9.8%，爆量% 由大到小）
  - `GET /api/dashboard/db-health` — 資料庫健康檢查（各表最新日期與筆數）

### 修改檔案
- `twse-app/src/server/app.ts` — 掛載 dashboard routes
- `twse-app/src/components/views/DashboardView.tsx` — 完全重寫
  - 移除加權指數卡片（保留指數數字顯示）、櫃買指數、漲跌分布、4 個舊排行榜
  - 新增 4 個 CollapsibleCard 區塊（預設收起）
  - 新增 DataSourceSelector 組件
  - 新增 Loading/Error/Retry 狀態處理

## 參數設定

| 查詢 | 過濾條件 | 排序 | 限制 |
|------|---------|------|------|
| 最近發放股利 | 近 7 天 dividend_events | 日期 ASC, 現金股息 DESC | 20 檔 |
| 投信連買二日 | trust_net >= 0 連續 2 天 | 成交量降冪 | 20 檔 |
| 突破 MA200 | 昨日收盤 < 昨日 MA200, 今日收盤 > 今日 MA200 | 收盤價降冪 | 20 檔 |
| 昨日涨停 | 漲幅 >= 9.8% | 爆量% 降冪 | 20 檔 |

## 已知問題

1. `break-ma200` 查詢對每檔股票個別計算 MA200（N 次查詢），資料量大時可能較慢。未來可優化為 CTE 批次查詢。
2. `foreign-buy-2day` 名稱仍沿用舊稱，實際是「投信連買」。未來可改名。
3. Supabase 查詢目前是 placeholder，實際邏輯未完整實作。
4. FinMind 僅支援單檔查詢，篩選功能不適用。
