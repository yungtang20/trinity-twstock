# TWSE 漲跌正負號修正 + 首頁顯示優化

## 本次任務
1. 修正 TWSE 漲跌點數正負號錯誤
2. 首頁顯示成交量和成交金額
3. 確認數據日期

## 執行過程

### 發現的問題

**問題 1: TWSE 漲跌點數正負號錯誤**
- API 回傳格式：`row[2]` = 符號 (`<p style='color:green'>-<\/p>`)，`row[3]` = 絕對值 (`1,057.05`)
- 原程式碼：直接讀 `row[3]` 作為 change，失去正負號
- 修正：先讀 `row[2]` 判斷正負，再乘以 `row[3]` 的絕對值

**問題 2: 首頁沒有顯示成交量和成交金額**
- API 回傳 `amount: 15307.83` (億元)
- DashboardView 沒有顯示這個數據
- 修正：在 Header bar 顯示成交金額

**問題 3: 數據日期確認**
- API 回傳 115年06月24日 數據
- 確認是最新的交易日數據

### 修正項目

1. **修正 TWSE 漲跌點數正負號**
   - 檔案：`src/server/services/twse.ts`
   - 新增從 `row[2]` 讀取符號邏輯
   - 修正前：change = +1057.05 (錯誤)
   - 修正後：change = -1057.05 (正確)

2. **首頁顯示成交量和成交金額**
   - 檔案：`src/components/views/DashboardView.tsx`
   - Header bar 改為並排顯示加權和櫃買指數
   - 每個指數下方顯示漲跌點數/百分比
   - 最下方顯示成交金額

3. **修正 OTC 漲跌百分比計算**
   - 檔案：`src/server/services/otc.ts`
   - 改用 API 回傳的 `row[6]` 而非自行計算
