## 1. 平台概述

TRINITY 台股分析平台是一個全方位的股票分析工具，提供即時行情、技術分析、籌碼分析、策略掃描和 AI 智慧分析功能。

### 核心功能
- **即時行情**：即時股票價格、漲跌幅、成交量
- **技術分析**：K 線圖、均線（MA5/25/60/200）、支撐壓力、POC、VSBC
- **籌碼分析**：法人買賣超、千張大戶、集保人數
- **策略掃描**：支撐壓力、均線、籌碼、型態、預測
- **AI 智慧分析**：9 種華爾街分析模板

### 目標用戶
- 台股投資者
- 技術分析愛好者
- 量化交易者

## 4. AI 分析數據收集與分析流程

### 4.1 數據收集策略

**數據來源優先級**：
1. FinMind API（免費會員）
2. Supabase
3. SQLite 本地資料庫

**FinMind API 數據集**：12 個免費數據集

**日期範圍**：2015-01-01 至今日（最大化）

### 4.2 單一報告生成流程

```
用戶點擊模板
    ↓
1. 驗證股票代號
    ↓
2. 收集 FinMind 數據
   ├─ 對每個數據集建立 API 請求參數
   ├─ 發送 POST 請求到 /api/finmind-proxy
   └─ 記錄結果（成功/失敗/無數據）
    ↓
3. 檢查數據是否足夠
   ├─ 如果無數據 → 返回「無數據」
   └─ 如果有數據 → 繼續
    ↓
4. LongCat 生成報告
   ├─ 建立提示詞（模板 + 股票代號 + 數據）
   ├─ 發送 POST 請求到 LongCat API
   │   ├─ model: LongCat-2.0
   │   ├─ max_tokens: 128,000
   │   └─ temperature: 0.7
   └─ 提取報告內容
    ↓
5. 顯示結果
```

### 4.3 LongCat API 請求格式

```json
{
  "model": "LongCat-2.0",
  "messages": [
    {
      "role": "system",
      "content": "你是一位專業的台股分析師..."
    },
    {
      "role": "user",
      "content": "分析需求：...\n\n股票代號：...\n\n可用數據：..."
    }
  ],
  "temperature": 0.7,
  "max_tokens": 128000
}
```

## 6. 技術棧

### 6.1 前端
- **框架**：React 18 + TypeScript
- **構建工具**：Vite
- **樣式**：Tailwind CSS
- **圖表**：lightweight-charts
- **動畫**：Framer Motion
- **圖標**：Lucide React

### 6.2 後端
- **運行時**：Node.js + tsx
- **框架**：Express
- **資料庫**：SQLite（better-sqlite3）+ Supabase

### 6.3 外部 API
- **FinMind**：台股數據
- **LongCat**：AI 分析
- **Supabase**：雲端資料庫
